Preview

报表预览

使用样例
var report = new TOONE.Report.Preview();
report.mount(document.getElementById('app'));
report.exportExcel("test.xlsx")

方法列表

exportExcel

导出excel

参数列表:
名称 类型 属性 描述
filename String

导出excel名称
options Object 可选

导出配置
ignoreFormula：忽略公式
ignoreStyle：忽略样式
返回值:

Promise

exportPdf

导出pdf

参数列表:
名称 类型 属性 描述
filename String

导出pdf文件名称
options Object 可选

导出配置
author：作者
creator：创建者
keywords：关键字
subject：主题
title：标题
sheetIndex：导出工作表下标，如未设置，则导出全部
返回值:

Promise

mount

报表挂载

参数列表:
名称 类型 描述
el DOMElement

挂载dom对象

print

报表打印，使用打印时，须在Preview初始化时将属性enablePrint设置为true

参数列表:
名称 类型 描述
params Object

打印参数
showBorder：显示excel外边框 showColumnHeader：显示列头 showRowHeader：显示行头 showGridLine：显示网格线 如未设置，则使用工作表内的配置值
返回值:

Promise

使用样例
var report = new TOONE.Report.Preview({
 enablePrint: true
});
var promise = report.print();
promise.then(function(){
 //打印成功
}).catch(function(e){
 //错误处理
});
参数列表:
名称 类型 描述
conf Object

配置信息
ready: Function<WorkBook> 初始化完成回调
enablePrint: Boolean 是否启用打印功能
json: Object 报表配置数据
license：String 报表许可证
